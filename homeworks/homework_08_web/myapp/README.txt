http://127.0.0.1:8000/ - главная страница, там есть классная менюшка. Можно зарегаться и авторизоваться. Сверху есть функции посмотреть все рейсы и какой-то конкретный(ссылка ссылается на первый id,поэтому если его нет то просто в адресе поставить нужный) 
После авторизации появляется метод POST сверху, а так же методы UPDATE и DELETE. Если зайти на эти страницы неавторизованным, то кинет на авторизацию. Если попробовать UPDATE или DELETE не свои записи, то выкинет сообщение по типу не ваша запись.
Список url:
http://127.0.0.1:8000/ - основная страница
http://127.0.0.1:8000/account/register/ - страница регистрации
http://127.0.0.1:8000/account/user_login/ - авторизация
http://127.0.0.1:8000/account/logout/ - выход
http://127.0.0.1:8000/api/flight/create/ - post по типу({"departure_time": "2019-02-12T22:14:00.000000Z", "arrival_time": "2019-02-12T22:18:50.000000Z", "airport": "hzgde", "aircraft": "hzkuda"})
http://127.0.0.1:8000/api/flights/list/ - все рейсы
http://127.0.0.1:8000/api/flights/1/detail/ - какой-то отдельный рейс
http://127.0.0.1:8000/api/flights/1/update/ - put
http://127.0.0.1:8000/api/flights/1/delete/ - delete
